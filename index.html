Welcome to VMSuperHub Statistics Collector.

This script allows you to collect statistics from from VMSuperHub modem and either send the results into carbon to be graphed via graphite or save it ti file in CSV format. The following metrics will be collected: power levels and SNR information.

Installation

To get up and running, install the required additional modules through pip. pip install -r requirements.txt

VMSuperHub Statistics is maintained by ldejager.
